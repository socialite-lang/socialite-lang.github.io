SociaLite

Easy, fast, and interactive query language for large-scale distributed graph analysis and data mining

  Install    Quick Start

Easy

Writing distributed code is easy with SociaLite. Its integration with Python makes it even fun to write distributed analysis code.

Fast

SociaLite programs are compiled to highly optimized parallel code. It is often as fast as manually implemented C code.

Interactive

SociaLite provides interactive command-line interface. It enables exploratory data mining and interactive debugging.
